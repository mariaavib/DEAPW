    • Archivo de Configuración Global

    El archivo de configuración global de Apache está ubicado en:

    • /etc/apache2/apache2.conf

    Este archivo contiene las configuraciones globales del servidor y aplica a todas las webs del servidor, salvo que estas directivas se redefinan en contextos inferiores como en Virtual Host.

    Variables de Entorno del Servidor Apache2

    Archivo de variables de entorno: /etc/apache2/envvars

    • valores que el servidor Apache utiliza.
    • Ejemplos de variables de entorno:
      • APACHE_RUN_USER
      • APACHE_RUN_GROUP
      • APACHE_RUN_DIR
      • APACHE_LOCK_DIR
      • APACHE_PID_FILE
      • APACHE_LOG_DIR

    Directivas Básicas

    • ServerName: Indica el nombre con el que el servidor se identifica a sí mismo.
      Ejemplo: ServerName daw27.deap27.net
    • ServerRoot: Directorio donde se encuentran los archivos de configuración de Apache.
      Ejemplo: ServerRoot /etc/apache2
    • ServerAdmin: Correo del administrador.
      Ejemplo: ServerAdmin admin@daw27.deap27.net
    • DocumentRoot: Directorio que contiene los archivos servidos por Apache.
      Ejemplo: DocumentRoot /var/www/html
    • PidFile: Fichero que almacena el ID del proceso de Apache.
      Ejemplo: PidFile /var/run/apache2/apache2.pid
    • User: Usuario con el que Apache se ejecuta.
      Por defecto: User www-data
    • Group: Grupo con el que Apache se ejecuta.
      Por defecto: Group www-data
    • ErrorLog: Archivo donde se almacenan los errores de Apache.
      Ejemplo: ErrorLog /var/log/apache2/errorlog.log
    • Listen: Dirección IP y puerto en el que escucha Apache.
      Ejemplo: Listen 80 y Listen 10.3.11.227:80
    • DirectoryIndex: Nombres de los ficheros servidos como índice.
      Ejemplo: DirectoryIndex index.html index.php
    • TypesConfig: Archivo con la lista de tipos MIME.
      Ejemplo: TypesConfig /etc/mime.types
    • DefaultType: Tipo MIME por defecto.
      Ejemplo: DefaultType text/plain
