    • Configuración Inicial de GIT

    git config --global user.name "Maria"➙ configurar el nombre y usuario
    git config --global user.mail "mariavidigalbarroso.guadalupe@alumnado.fundacionloyola.net"➙ configurar el correo
    Esto asegura que tanto el nombre como el correo estén configurados para todos los commits
    git --version ➙ Este comando se utiliza para verificar la versión de Git que está instalada en tu sistema.
    git config --list ➙ muestra una lista de todas las configuraciones de Git que están actualmente establecidas en tu entorno.

    Algunos comandos básicos

    git help ➙ muestra la ayuda del comando que se le indique
    git init ➙ inicializa el repositorio local

    Subir cambios

    • git init ➙ Para crear nuestro staging (área de preparación) y repositorio local
    • GIT ADD agregamos archivos al área de staging con:
      • git add nombre_archivo ➙ para agregar un archivo específico
      • git add . ➙ para agregar todos los archivos
    • git commit ➙ enviamos nuestroa archivos al repositorio local. Agregando -m "" podemos colocar un mensaje y registrar los commits realizados
    • git push ➙ enviamos nuestros cambios al servidor remoto
