    Evaluación del progreso

    Que estoy aprendiendo

    Lo que he aprendido aquí ha sido básicamente a usar GIT, ya que no sabía manejarlo. Ahora entiendo mejor cómo crear y gestionar repositorios, realizar commits para guardar los cambios, y utilizar ramas para trabajar en diferentes características o correcciones de manera aislada. También he aprendido a fusionar ramas y resolver conflictos que puedan surgir durante el proceso de fusión. Además, he comprendido la importancia de los comandos git fetch y git pull para mantener mi repositorio local actualizado con los cambios del repositorio remoto.

    Que recursos estoy utilizando

    Los recursos que estoy utilizando son las presentaciones que tenemos en classroom, aprender de las explicaciones de clase...

    Que dificultades estoy encontrando

    A veces, al fusionar ramas, encuentro conflictos que no siempre son fáciles de resolver. Algunos comandos de GIT como por ejemplo el GIT merge al fusionar los cambios de una rama a otra.
